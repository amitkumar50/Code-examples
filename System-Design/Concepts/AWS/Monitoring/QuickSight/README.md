### QuickSight
This service collects information from various sources as aws sources, spreadsheets, Big data sources, 3rd party DBs and build dashboards and UI for analysis.
```c
  onprem or cloud
  aws sources ---------data-------\
  spreadsheets  -------------------\
                                  QuickSight Dashboard(ML powered)
  Big data sources-----------------/
  3rd party DBs ------------------/
```
