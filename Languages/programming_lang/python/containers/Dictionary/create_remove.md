## 1. Dictionary Creation
### 1A. Using Initialization
```
dic = {'a':10, 'b':20}
dic['c']=30
print(dic)                #{'a': 10, 'b': 20, 'c': 30}
print(dic['a'])           #10
print('a' in dic)         #True
```

### 1B. Using dict() constructor
```
f = dict([('ab', 4), ('cd', 5), ('ef', 6)])
print(f)                            #{'ab': 4, 'cd': 5, 'ef': 6}
```
