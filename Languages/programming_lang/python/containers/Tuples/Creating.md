## Creating Tuples

### 1. Create tuple after taking input from user
```
a = input("Input some comma seprated numbers : ")     #1,2,3,4
t = tuple(a)
print('Tuple:',t)                                     #(1,2,3,4)
```

### 2. Created with or without `()`
```
b = (1,3,'f','my')
a = 1,3,'f','my'
print(a)                #(1, 3, 'f', 'my')
```
