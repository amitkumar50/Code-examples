## QUIC / Quick UDP Internet Connections 
- In 2012, Google started developing QUIC. 2014 QUIC is used on all Google services, Chrome, and mobile apps.
- QUIC takes
  - Reliablilty from TCP.
  - Fast/Rapidity from [UDP](../UDP).
- **QUIC vs TCP**
```html
  HTTP/2          HTTP/2
  TLS 1.2         QUIC
  TCP             UDP
  IP              IP
```
