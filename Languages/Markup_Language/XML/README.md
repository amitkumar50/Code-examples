## Example
```xml
<?xml version="1.0" encoding="UTF-8"?>        //XML Version
<note>                                        //Root Element
  <to>Tove</to>                               //Child Elements: to, from, heading, body
  <from>Jani</from>
  <heading>Reminder</heading>
  <body>Don't forget me this weekend!</body>
</note>                                        //Root Element End. Closing Tag is Must
```
