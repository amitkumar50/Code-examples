```rs
       Function          desc                   Expects
-------------------|------------------------|------------
  assert!(a)       | if (a==false) assert   |  true
 assert_eq!(a, b)  | if (a!=b) assert       |  equal
 assert_ne!(a,b)   | if (a==b) assert       |  not equal
```
