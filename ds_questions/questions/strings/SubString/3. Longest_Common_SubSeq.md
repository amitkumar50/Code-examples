### Problem
- Find longest common subsequence in 2 strings.
- Example
```c++
  input str1 = "sea"
  input str2 = "eat"
  Answer = 2 "ea"
```

## Logic-1 (Recursive)
```c++
                sea, eat
                /       \
            sea,ea      se,ea
            /           /   \
          se,e        se,e   s,ea
          /           /       \
         s,0        s,0       0,a
See repeated calculation of "se,e"
And in bigger string these repeated recurions will be more.
```
### Complexity
- 2<sup>n</sup>
### Code-1(Recursion)
```c++
int lcs(string& a, string& b, int m, int n){
  if (m == 0 || n == 0)
    return 0;

  if (a[m-1] == b[n-1])
    return 1 + lcs(a, b, m-1, n-1);
  else
    return max(lcs(a, b, m, n-1), lcs(a, b, m-1, n));
}
int main(){
  string a = "sea";  string b = "eat";
  cout<<"Length of LCS is "<< lcs(a, b, a.size(), b.size()) ;     //2
}
```
