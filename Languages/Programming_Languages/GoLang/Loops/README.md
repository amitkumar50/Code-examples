### for loop
#### Basic for loop
```go
 for initialization; condition; increment/decrement {
    // Code to be executed
 }

for i := 1; i < 10; i++ {
}
```

<a name=w></a>
### while
There is no while in go, use for instead
```go
for i < len(arr) {
 ...
 i++;
}
```
