### SRX100 vs SRX110

||SRX100|SRx110|
|---|---|---|
|VDSL/ADSL2+(integrated WAN port)|No|Yes|
|Memory Model|High memory & Low memory both|Only High Memory|

### Memory Model
- **Low Memory:** 512 MB
- **High Memory:** 1 GB
