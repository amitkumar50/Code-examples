BFS / Level Order Traversal

### Level Order Traversal
Tree is traversed level by level.
```c
Graph: 	0 -- 1 -- 2		//Output: 0 1 3 2
         \   |	 /
	           3
Tree:
        12
       /  \
      10  30
     /    / \
   14   25   40
             /
            50
O/P ->  12 
        10 30 
        14 25 40 
        50
```
