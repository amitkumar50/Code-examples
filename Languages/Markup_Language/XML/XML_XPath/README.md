## XML XPath
- **What?** Complete path of element in XML document.
- **Example**
  - XPath of 1st book=`/bookstore/book[0]`
```xml
<?xml version="1.0" encoding="UTF-8"?>

<bookstore>

<book category="cooking">
  <title lang="en">Everyday Italian</title>
  <author>Howdy</author>
  <year>2005</year>
  <price>30.00</price>
</book>

</bookstore>
```
