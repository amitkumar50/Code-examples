## Web Service
- **What?** Web service is process/service running inside web server(hosted over cloud) reachable at port on web server, that provides any information needed. Eg: Whether report.
```c
    Web Client                                  {Web Service} On US Server
  rest-api call() ----Give me Temp of NY------>    |
                 <-NY temp (JSON or XML format)--- |
```
